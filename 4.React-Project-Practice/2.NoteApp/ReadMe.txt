This project was initialized in React 17 instead of 18 for compatibility

The following commands need to be ran:

1.
npm create vite@latest

2.
cd notesapp

3. 
npm i react@17 react-dom@17


These are the dependencies download and their npm commands

1.React MarkDown editor
npm i react-mde

2.Nano ID generator
npm i nanoid

3. React Split for pane resizing
npm i react-split 

4. React Showdown for rendering markdown as React components
npm i react-showdown
